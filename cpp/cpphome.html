  • C++ is a general-purpose programming language. It has imperative,
    object-oriented and generic programming features,
    while also providing facilities for low-level memory manipulation.

  • C++ was designed with a bias toward system programming and embedded,
    resource-constrained and large systems, with performance,
    efficiency and flexibility of use as its design highlights.

  • Many other programming languages have been influenced by C++,
    including C#, D, Java, and newer versions of C.
